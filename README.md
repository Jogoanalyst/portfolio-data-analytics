## Proyectos Destacados

### 1. Análisis Predictivo del Riesgo de Enfermedad Coronaria
Modelo de clasificación para predecir el riesgo de enfermedad coronaria a 10 años utilizando datos del estudio de Framingham.

**Principales tareas realizadas**
- Limpieza y análisis exploratorio de datos (EDA)
- Tratamiento de desequilibrio de clases con SMOTE
- Modelado con Regresión Logística
- Evaluación con métricas de clasificación (Recall, F1-score)

**Resultados**
- Mejora del recall en clase positiva de 4% a 63%
- Identificación de variables relevantes: edad, presión sistólica y colesterol

**Tecnologías**
Python, Pandas, Scikit-learn, Matplotlib, Seaborn, Imbalanced-learn

### 2. Predicción del Precio de Vuelos
Modelo de regresión para estimar precios de vuelos a partir de variables operacionales y comerciales.

**Principales tareas realizadas**
- Integración y limpieza de datasets
- Feature engineering en variables temporales
- Entrenamiento y comparación de modelos de regresión
- Evaluación con métricas MAE, MSE y R²

**Resultados**
- Modelo Random Forest seleccionado como mejor desempeño
- R² = 0.99

**Tecnologías**
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
